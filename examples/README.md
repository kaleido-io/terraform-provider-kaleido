## Examples

This directory contains resource-specific examples of how to use the Terraform provider for the Kaleido Enterprise Platform for the documentation, as well as
older examples of using the (deprecated) Kaleido Blockchain-as-a-Service resources.

For modules and use case examples of the Kaleido Enterprise Platform, please see the [official Terraform modules](https://github.com/kaleido-io/terraform-kaleido-modules) repository.
